哲学

一人一人が創造性を磨く。

  1. 創造性が価値を生み出す。
  2. 一人一人が持つ素晴らしい可能性をもっと実感できる現実を創る。
  3. 一人一人が未来にワクワクする気持ちを持てるような現実を創る。
  4. 一人一人が持つ創造性を集め、全員の達成感に繋げる。

一人のためにみんなが協力をし合う関係。

  1. ワクワクする未来を実現する。
  2. 人生は、ハッピーエンドのストーリー。
  3. 人は誰でも、本気でやりたいと願った事ことなら実現できる。

ビジョン

  1. 価値観、考え方が違う人間同士でも協力しあうことで、全員が達成感を感じる事ができる現実を作りたい。
  2. 一人ではできないことでも、みんなで力をあわせることで、成し遂げられるという達成感を共有したい。
  3. 多くの人が実現している、という体験を見て、「自分にもできるかもしれない」と感じて欲しい。

コンセプト

  1. シンプル。
  2. いつでも、どこでも、見られる。
  3. 物語で伝える。
  4. 実名性
  5. フィードバック

モデル

  1. スマートフォン
  2. ユーザーはそれぞれ、プロフィールにワクワクすること、得意な事、好きな事を書く。
  3. サポーターは、応援ボタンを押して、スターターをフォローすることで応援する。
  4. サポーターは、コメントやアドバイスをすることで、スターターを応援する。
  5. サポーターの役割は、スターターが１歩でも前に進むように「行動」を促すこと。
  6. スターターは、「行動」をし、そのときの状況や心境を書くことで、サポーターに状況を伝える。
  7. オブザーバーは、スターターでもサポーターでもない、第3者。

サービスサイクル(サービスとして回る仕組み)

  1. スターターがビジョンを設定する。
  2. オブザーバーは、スターターが書いたビジョンを見て、応援ボタンを押して、サポーターになる。
  3. サポーターが生まれる。

ベネフィット

スターターにはどんなメリットがあるの？

  1. 「目標を宣言することで、応援者が集められる」体験。
  2. 目標を達成するために必要な情報を、サポーターから提供してもらえる。
  3. 自分が目標を達成できる人間だということを実績としてアピールできる。

サポーターにはどんなメリットがあるの？

  1. 自分が共感した目標を実現するまでの過程にリアルタイムに参加できる。
  2. 自分ができる些細な事でも、人を助ける事ができるという体験。
  3. 自分がいずれ、スターターになったとき、過去自分が助けたスターターの人から応援してもらえる。

オブザーバーにはどんなメリットがあるの？

  1. 前向き、未来志向な人たちが集まっている場所に来ることで、普通の場所とは違う体験をすることができる。
  2. みんながそれぞれ自分のやりたい事を頑張っているのを見て、前向きなエネルギーをもらえる。
